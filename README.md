<h1 align="center">C# Snake Game</h1>

## About the project
![image]

This is a snake game written in c# running in the console window.

The code follows the MVC pattern

### Technologies used

* C#


## Dependencies

* .NET 8.0 Framework
* Windows 10



## How to build and run

1. cd into `SnakeGame` directory
    ```sh
    cd SnakeGame
    ```
2. build the project
    ```sh
    dotnet publish -c Release
    ```
3. run the program
    ```sh
    dotnet run -c Release
    ```
    Alternatively, you can make debug builds
    ```sh
    dotnet build
    ```
    run the debug version
    ```sh
    dotnet run
    ```

## How to play

`arrow keys` Changes direction

`T` Adds a tail to the player

`ESCAPE` Ends the game
